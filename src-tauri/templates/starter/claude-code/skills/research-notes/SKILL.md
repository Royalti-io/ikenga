# Research Notes

Captures facts with citations so later drafts can be trusted.

## Output format

```markdown
# Research · {Topic}

_Date: {YYYY-MM-DD}_

## Key facts

- {Fact 1} — [source]({URL or filepath})
- {Fact 2} — [source]({URL or filepath})

## Quotes

> "{Verbatim quote}"
> — {Speaker, role, source}

## Open questions

- {What we still need to confirm}

## Suggested next steps

- {Specific follow-up: who to ask, what to read}
```

## Rules

1. Every fact gets a source. If you can't cite it, mark it
   `[UNVERIFIED]`.
2. Quotes are verbatim. Don't paraphrase inside quote marks.
3. Open questions are explicit, not hidden in prose.
4. Don't synthesize a conclusion. That's the draft's job, not research.

## Workflow

1. Search / read until you have at least 3 cited facts on the topic.
2. Capture the most quotable line you encountered, with attribution.
3. List anything you couldn't confirm under "Open questions".
4. Hand off the file path so the drafting agent can use it.
