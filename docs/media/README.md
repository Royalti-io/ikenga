# README media assets

Screenshots and GIFs referenced by the top-level `README.md`. Drop the captures
here with the exact filenames below and they render automatically.

| File | Type | What it should show |
|---|---|---|
| `hero.png` | screenshot | Full app window: the Home canvas with a couple of widgets, a chat pane open on the right. The lead image — make it representative and clean. |
| `pkg-dev.gif` | gif | `ikenga dev <path>` mounting a pkg, then an edit-on-save triggering an in-place reload in the running shell. Keep it short (~5–8s loop). |
| `chat-engine-picker.png` | screenshot | The two-level Engine → Model popover open in the chat composer. |
| `packages.png` | screenshot (optional) | The pkg manager / browse view. |

Guidance:
- Capture on a clean profile (no personal data in panes).
- PNGs: trim to the window, ~2x for retina sharpness.
- GIFs: keep under ~3 MB; prefer a tight crop on the relevant region.
