Ikenga · Design / Rung 1 · Token System

Tokens, end to end.

Three themes × two modes = six surfaces. Workspace tints stay constant across themes (Mail = amber, Outbox = ember, Studio = oxblood, Agents = verdigris). Typography, spacing, radius, elevation, motion are theme-agnostic. Flip the controls above to live-test any combination — every section below reacts.

02 · Color · Semantic accents

Four semantic roles. Each theme assigns its own hue.

Roles stay constant across themes: primary = "the right hand" / decisive action · achievement = trophy, done, won · danger = warnings, overdue, destructive · systemic = agents, observability, calm. In Theme C the primary becomes verdigris and danger (oxblood) takes the action-only role — note how the meaning shifts but the slots don't.

--primary

--achievement

--danger
Overdue, destructive, warning.
--systemic
Agents, observability, calm.

03 · Color · Workspace tints (constant)

Eight tints that don't move when the theme moves.

06 · Radius

Six radii. Every chrome surface lands on one.

07 · Elevation

Four levels. Light and dark have separate values — never reuse.

The cockpit is mostly flat. Elevation is reserved for things that genuinely float: popovers, command palette, dialogs, dragged items. If you find yourself reaching for shadow-3+ on a static element, the answer is usually a stronger border instead.

--shadow-0
Static surface. The default. No floating.
--shadow-1
Hairline shadow. Rests on, doesn't float.
--shadow-2
Hover lift, dropdown, tooltip.
--shadow-3
Popover, sheet, command palette.
--shadow-4
Modal, dragged item. Sparingly.
08 · Motion

Click any track to fire. Three durations × three easings.

fast for state-only (hover, focus). base for component-level (tab switch, list reorder). slow / slower only when something is moving across the layout (sheet open, mode swap). calm is the default ease — gentle in, soft out. decisive for committed actions (send, confirm). bounce only for celebration moments — never for navigation.

09 · Density

Three densities. The same content, three ways to live with it.

Compact = Linear-tight, fits more on screen, smaller hit targets. Comfortable (default) = the canonical sweet spot. Spacious = for longer sessions, larger type, breathing room. Density only affects row heights, body sizes, and gap units — surface tokens, accent tokens, typography scale remain identical.

10 · Cheat sheet

What goes where.

When in doubt, follow this map. The token you reach for is rarely the question — the question is which role is at play.

When something is decisive
Use --primary. Reply CTA. Active nav indicator. Unread tick. The "do this now" colour.
↳ at most one per screen region
When something is earned
Use --achievement. Won deals. Completed states. Hot-thread counts. Toast on success.
↳ celebrate, but don't shout
When something is overdue
Use --danger. Past-due tasks. Failed deploys. Destructive confirms. Never decoratively.
↳ if everything is danger, nothing is
When something is just running
Use --systemic. Agent status. System notifications. Calm observability data.
↳ infrastructure colour, low pulse
