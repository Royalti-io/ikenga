The chrome the user lives in.

Click any icon in the activity bar to swap workspaces — sidebar replaces, content tabs reskin, workspace tint shifts. The right-edge dock pane is global and collapsible (rail when minimized, full pane when expanded), and tabs flow both ways between content and dock.

Live shell · click activity bar to switch workspace · resize controls above
⓵ ACTIVITY BAR
56px · 7 + 2 + Settings
App / Mail / Outbox / Studio / Agents / Files / Sessions, plus Canvas Design and Image Generator (mini-apps), plus Settings at the bottom. Active indicator uses --tint-{ws}-fg. Horn glyph is the brand mark.
⓶ SIDEBAR
8–30%, mode-aware
Each workspace owns its sidebar. Top header carries a faint --tint-{ws}-bg wash so muscle memory tells you where you are. Toggle Collapsed / Hidden in the controls — collapsed shows 90px stub.
⓷ CONTENT + TABS
Splittable, draggable
Section tabs at top can be pinned, closed, dragged. Drop into edges to split, into another tab strip to combine. The route outlet renders below.
⓸ DOCK PANE
Global · collapsible · drag-friendly
Right edge. Persists across workspace switches. Holds Chat / Terminal / Artifact tabs by default. Drag tabs in/out to reorganize. Toggle in the controls above.
Workspace tints in action

Tints are constant hues across themes — Mail is amber whether you're on Theme A, B, or C. Mode (light/dark) modulates only lightness. The activity-bar active indicator, sidebar header gradient, and section-tabs underline all share the tint, so changing workspace feels coherent without making the chrome loud.
